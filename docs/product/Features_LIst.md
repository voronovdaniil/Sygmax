# Sygmax Databases - создание и управление базами данных

## Основные возможности

- **Типы представления:** таблица, галерея, канбан-доска, календарь, диаграммы.
- **Гибкость интерфейса:** возможность легко переключаться между видами.
- **Фильтрация:** гибкие фильтры по нескольким параметрам.
- **Сортировка:** по любой колонке или пользовательскому алгоритму.
- **Связь между базами данных:** возможность связывать записи (например, связь задач с проектами, людьми или документами).
- **Отображение связанных данных:** представление всех связанных данных в одном интерфейсе.

## Формулы и расчёты

- Поля для вычислений (например, сумма значений, разница между датами).
- Возможность использования пользовательских формул (аналог Excel).

## Автоматизация данных

- Сценарии автоматического заполнения (например, если дата X истекла, изменить статус на "Просрочено").
- Автоматическое обновление связанных данных.

## Настраиваемые шаблоны записей

- Возможность создавать шаблоны для повторяющихся типов данных (например, стандартный шаблон для задач или контактов).

## Интеграция с календарём

- Автоматическая синхронизация дат из базы данных с внешними календарями (Google, Outlook).

## Импорт и экспорт

- **Импорт:** Excel, CSV, JSON.
- **Экспорт:** Excel, PDF, JSON, API для интеграции.

## Типы данных для полей

- Текст, числа, даты, флажки, списки, ссылки, изображения, файлы, формулы, отношения (relational), прогресс-бары.
- Поддержка вложенных объектов (например, таблица внутри записи).

# Sygmax Feeds -- RSS-клиент для потребления и управления внешними источниками контента

## Основные функции RSS-клиента

- **Поддержка стандартных форматов:** RSS и Atom.
- **Импорт и экспорт:** подписок в формате OPML.
- **Добавление источников:** вручную или по URL.
- **Обновление фидов:** автоматическое с заданным интервалом.
- **Настройка частоты обновления:** индивидуально для каждого фида.
- **Группировка:** фидов по категориям.
- **Теги:** возможность назначать теги фидам.
- **Поиск:** по источникам и заголовкам.
- **Просмотр:** полной статьи прямо в клиенте.
- **Уведомления:**
  - По email.
  - Через Telegram или другие мессенджеры.

## Интеграция с заметками

- **Сохранение фрагментов** RSS-статей в заметках.
- **Автоматическая ссылка** на оригинальную статью.
- **Теги из RSS** используются как теги в заметках.
- **Привязка статей** к существующим заметкам.
- **Вставка мультимедиа** из статьи прямо в заметку.
- **Автоматическое создание заметок** для избранных статей.
- **Комментарии** к RSS-записям в заметках.

## Интеграция с базами данных

- **Импорт метаданных:** заголовок, автор, дата.
- **Сортировка:** по дате, источнику, популярности.
- **Категории:** создание отдельных баз данных для разных фидов.
- **Связь с записями** в существующих базах данных.
- **Отслеживание изменений** в RSS-записях.

## Расширенные возможности фильтрации и сортировки

- **Настраиваемые фильтры** по ключевым словам.
- **Исключение источников** или записей из ленты.
- **Сортировка:** релевантность, длина текста, количество мультимедиа.
- **Подсветка новых статей**, соответствующих ключевым словам.
- **Удаление старых записей** через заданный период.

## Работа с мультимедиа

- **Просмотр изображений, видео и аудио** прямо в клиенте.
- **Скачивание мультимедиа** для офлайн-доступа.
- **Создание галерей** из изображений внутри статьи.
- **Воспроизведение подкастов** прямо в клиенте.

## Совместная работа

- **Совместное использование статей** с другими пользователями.
- **Совместное комментирование** в реальном времени.
- **Общие категории** RSS-источников для команды.
- **Отмечание статей как "прочитанных"** для всех участников.

## Автоматизация

- **Автозаполнение заметок** из RSS-статей по шаблонам.
- **Создание напоминаний или задач** из статей.
- **Связь с API** для автоматизации процессов.

## Настраиваемость

- **Темы оформления:** светлая/тёмная.
- **Перемещение панелей** внутри интерфейса.
- **Горячие клавиши** для быстрого доступа.
